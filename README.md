# CMPUT307-Lab07
# Group Project
## Project Overview
This project will be completed in three labs and conducted in groups of three. The goal
is to create a short one-minute animation that demonstrates your understanding and
integration of all the materials and techniques covered in Labs 1–6. The animation
should combine human/face modeling, still life, texture application, adaptive lighting,
and a cohesive background environment.
Group Project (part 1) – Modeling and Scene Design
To build all the visual assets for your animation, including the human/face model and
still life objects.
## Tasks:
1. Human/Face Model (40 marks, 20 marks for each task)
○ Model or import a human or face mesh.
○ Prepare it for animation (rigging, blend shapes, or motion capture setup).
2. Still Life (60 marks, 30 marks for each task)
○ Design a small still life composition (minimum three objects).
○ Apply three different textures using material maps.
